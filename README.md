# AK_Landing_Page_BS

landing page using bootstrap 5 and sass 

live - preview :

![preview](src/imgs/preview.gif)

## Technologies
The Application using of the following libraries:
- Webpack 5
- Bootstrap 5
- SASS
- Glightbox
- Animate.css

## Getting Started

### 1. Install the dependencies :
```sh
npm i
```
### 2. Use webpack - Production :
```sh
npm run prod
```
### 3. Open - index.html :
```sh
./dist/index.html
```
#### For Development :
```sh
npm run dev
```
